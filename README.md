# Sketch Assistant Generator

## Description

A dynamic Sketch Assistant Generator API, built on NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
