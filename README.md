# Sketch Assistant Generator

<a href="https://sketch-assistant-generator.herokuapp.com/api/">Visit API</a>

## Why this exists?

Creating a Sketch Assistant, and defining the rules that you want in it requires knowledge of JavaScript and NPM.

This project's goal is eliminating the need to create assistant packages manually and to expose and extend multiple configurable rules and assistants.
