# Sketch Assistant Generator

<a href="https://sketch-assistant-generator.herokuapp.com/api/">Visit live</a>

## Description

A dynamic Sketch Assistant Generator API, built on NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

# e2e tests
